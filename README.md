## Prerequisites

Before you begin, ensure you have met the following requirements:

- **Ruby** (version 3.0+)
- **Rails** (version 7.0+)
- **Node.js** (version 18.0+)
- **Yarn** (version 1.22+)

You can verify your versions by running:

```bash
ruby -v
rails -v
node -v
yarn -v
